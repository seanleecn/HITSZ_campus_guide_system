# 哈尔滨工业大学（深圳）校园导游系统

数据结构与算法分析(C++)这门课的课程作业

## 实现了三个功能：

1. 使用无向图表示地图，提供**景点查询**功能

2. 输入起点、终点、途径的景点个数；输出满足上述条件的全部路径中最短的一条
3. 输入起点；输出从该点到其他点的最短路径，并按照路径长度排序

## 使用的方法

	1. 邻接矩阵存储无向图
 	2. DFS遍历图
 	3. Floyd算法找到全部路径点+快速排序对路径进行排序
